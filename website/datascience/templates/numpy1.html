NumPy Mean

The mean function calculates the arithmetic mean (average) of elements along a specified axis.

NumPy Sum

The sum function computes the sum of array elements over a specified axis.

NumPy Standard Deviation

The std function calculates the standard deviation of elements along a specified axis.

NumPy Max

The max function returns the maximum value along a specified axis.

NumPy Min

The min function returns the minimum value along a specified axis.

NumPy Transpose

The transpose function returns the transpose of the array. It reverses or permutes the axes of the array.
